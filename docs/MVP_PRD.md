# Andy'K Landing Page — MVP PRD (AI-Actionable)

**Project:** andyk-landing
**Stack:** Next.js 16 + React 19 + TypeScript 5 + Tailwind CSS v4
**Design Reference:** `docs/DESIGN_PLAN.md`
**Task Board:** `docs/KANBAN.md`

---

## Current State

The landing page is a single-page Next.js app at `/` with 14 active components. It has:
- Hero with 2 CTAs
- Scrolling top banner
- Navbar with hash navigation
- 6-step A.D.A.M. roadmap pipeline visualization
- Founders section (CEO + CTO with photos, bios, LinkedIn)
- 3-tab pricing section (B2B / B2G / Tech — 13 plans total)
- Social proof section (stats: 150+ campaigns, etc.)
- FAQ accordion
- Integrations section
- Dark-bg CTA section
- Footer with links + company info

### What's Missing
1. `/onboarding` route — referenced by 6 CTAs, does not exist
2. Contact form — all CTAs are mailto links
3. SEO metadata — no Open Graph, no Twitter cards, no structured data
4. Image optimization — co-founder.jpg is 103KB unoptimized
5. Dead code — 5 unused Mock components
6. Analytics — no tracking at all
7. Playwright tests — only 1 example test exists
8. Sitemap + robots.txt — missing for SEO

## Feature Specs

### F1. Contact Form Component

**Priority:** HIGH — replaces all mailto links with a real form

**Requirements:**
- React server action or API route for form submission
- Fields: Name, Email, Company, Service Interest (dropdown: B2B/B2G/Tech/A.D.A.M. License), Message
- Email validation (company-domain preferred, public allowed for landing page)
- Success/error states with clear feedback
- Mobile: full-width inputs, 44px height, sticky submit button
- Desktop: 2-column layout (info left, form right)
- Send email via EmailJS or Resend API (configurable)
- Honeypot + rate limiting (no captcha)
- Add to CTA section and as standalone `#contact` section

**Design:**
- White card on dark background (CTA section)
- Inputs: `border-grid-500`, `focus:border-highlight`, rounded-lg
- Submit button: `bg-highlight text-white` or `btn-primary-gradient`
- Success message: highlight-colored with checkmark

**Data model addition to `data.ts`:**
```typescript
export const CONTACT_FORM = {
  heading: "Let's Talk",
  subheading: "Tell us about your business and we'll show you how we can help.",
  serviceOptions: [
    "B2B Business Development",
    "B2G Public Sector Advisory",
    "Technology / CTO Services",
    "A.D.A.M. License Inquiry",
    "Partnership",
    "Other",
  ],
};
```

### F2. `/onboarding` Route

**Priority:** HIGH — 6 CTAs point here, returns 404

**Requirements:**
- New route at `src/app/onboarding/page.tsx`
- Simple landing page that explains what happens next
- Embedded questionnaire or link to external questionnaire
- Phase 1: Static page with "Coming Soon" + contact form fallback
- Phase 2: Typeform-style questionnaire (when A.D.A.M. backend is ready)

**Design:**
- Same layout as homepage but simplified (Navbar + content + Footer)
- Hero area with "Tell Us About Your Business" heading
- 3 cards showing the 3 service types (B2B/B2G/Tech) with brief descriptions
- Each card links to mailto with pre-filled subject OR future questionnaire
- Bottom CTA: "Not sure? Contact us directly" → contact form

**Route structure:**
```
src/app/onboarding/
  page.tsx       # Main onboarding landing
  layout.tsx     # Optional: shared layout with Navbar/Footer
```

### F3. SEO & Meta Enhancement

**Priority:** MEDIUM — needed before deployment

**Requirements:**
- Open Graph meta tags (title, description, image, url, type)
- Twitter Card meta tags (summary_large_image)
- JSON-LD structured data (Organization + WebSite)
- Dynamic `<title>` per route (homepage vs onboarding)
- Canonical URLs
- `sitemap.xml` generated by Next.js
- `robots.txt` with sitemap reference

**Implementation in `layout.tsx`:**
```typescript
export const metadata: Metadata = {
  metadataBase: new URL("https://andykgroupinternational.com"),
  title: {
    default: "Andy'K Group International | B2B & B2G Business Automation",
    template: "%s | Andy'K Group International",
  },
  description: "Strategic partner in business automation & outreach...",
  openGraph: {
    type: "website",
    locale: "en_GB",
    siteName: "Andy'K Group International",
    images: [{ url: "/og-image.png", width: 1200, height: 630 }],
  },
  twitter: {
    card: "summary_large_image",
  },
};
```

**Create:**
- `public/og-image.png` — 1200x630 branded Open Graph image
- `src/app/sitemap.ts` — Next.js sitemap generator
- `src/app/robots.ts` — Next.js robots.txt generator
